# Depth vs Width in Multilayer Perceptrons: Understanding Expressivity

This repository contains the code, figures, and final report for an analysis of how **depth** and **width** affect the expressivity of Multilayer Perceptrons (MLPs). The project explores theoretical foundations, trains multiple neural network architectures, and visualises decision boundaries, accuracy trends, and parameter efficiency.

The work was completed as part of an individual assignment for the University of Hertfordshire.
